Formulario

Perguntas e respostas

  1. Na tag "input" com atributo type="text", para que serve o atributo "placeholder"?
    2. R: O atributo placeholder especifica uma dica curta que descreve o valor esperado de um campo de entrada
    (por exemplo, um valor de amostra ou uma breve descrição do formato esperado). A dica curta é exibida no campo de entrada antes que o usuário insira um valor.

  2. Na tag "input" com atributo type="radio" ou type="radio", para que serve o atributo "checked"?
    3. R: especifica que este botão "checkbox" ou "radio" é selecionado por default.

  3. Na tag "input" para que serve o atributo "value"?
    4. R: O value atributo especifica o valor inicial para um campo de entrada:

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
    5. R: (name) O mesmo nome

  5. Para que serve a tag "label"? Para que serve o atributo "for" nessa tag?
    6. R: é usada para criar rótulos para itens em uma interface de usuário. O atributo "for" faz um tipo de link entre a label e o input . Logo o for tem que apontar para o ID do input .

  6. Na tag "option" para que serve o atributo "selected"?
    7. R: Quando presente, especifica que uma opção deve ser pré-selecionada quando a página for carregada. A opção pré-selecionada será exibida primeiro na lista suspensa.

  7. Nas tags de formulário para que serve o atributo "required"?
    8. R: é usado para indicar que a entrada do usuário é obrigatória em um elemento antes que um formulário possa ser enviado.

  8. Na tag "form" para que serve o atributo "action"?
    9. R: define o local (uma URL) em que os dados recolhidos do formulário devem ser enviados.

  9. Na tag "form" para que serve o atributo "method"?
    10. R: O atributo method especifica como enviar dados de formulário(get ou post)

  10. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag "form"?
    11. R: Method="post"

  11. Qual valor deve ser informado no atributo "method" na tag "form" em um formulário que será usado para fazer upload de arquivos?
    12. R: Method="get"
